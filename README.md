# Python Challenge Solutions
### Simple program that provides answers to the Python Challenge by Nadav Samet, in addition to stepping users through each challenge. 
___
# Set UP
### 1) Clone repo or download files.
### 2) pip install requirements from requirements.txt
### 3) Run the program.
##### NOTE: Presentation is better in a larger window so I would recommend running in terminal and making it full screen
___
